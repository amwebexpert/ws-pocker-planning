# WSPockerPlanning

Server for handling agile pocker planning sessions communications through WebSocket clients

## Build and start

    npm run clean
    npm run build
    npm start

### Localhost deployment

- ws://localhost/ws

### Production deployment(s)

- ws-poker-planning.onrender.com
